WHAT IS DOM(Document Object Model):-

In JavaScript, the DOM (Document Object Model) is a representation of a web page that allows JavaScript to interact with and modify the content and structure of the page. It provides a way to access, manipulate, and update HTML elements on a webpage using JavaScript code.
